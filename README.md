# Sentinel-Shredder
hybrid ai security tool using Google Gemini to manage file risk and a C++ engine for multi pass secure data destruction (used python as well)
*suppose u want to delete a file so the first thing that it does is the C++  backend will get the file and then after getting it it will use the python libraries where AI is used to categorize the priority of the file like 1. 2 3. and then file will be sent back to the C++ backend and then it will shredd the fil according to the priority
